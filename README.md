# 🏆 상식퀴즈대전 (Trivia Battle Arena)

실시간 멀티플레이 상식 퀴즈 서바이벌 웹 서비스입니다.

## 📌 주요 특징
- **실시간 대전**: BroadcastChannel 기반 탭/기기 간 실시간 동기화 대전
- **10문제 1세트**: 1~9번(1점) + 10번(2점 골든벨 역전 퀴즈)
- **100+ 검증 상식 DB**: 도전 골든벨 유형의 검증된 양질의 주관식 상식 퀴즈
- **스마트 채점**: 띄어쓰기, 대소문자, 숫자/단위 자동 유연 정규화 채점
- **최종 시상대**: 1, 2, 3위 3D 단상 및 축하 효과

## 🚀 개발 및 실행
```bash
# 의존성 라이브러리 설치
npm install

# 개발 서버 구동
npm run dev

# 프로덕션 빌드
npm run build
```
